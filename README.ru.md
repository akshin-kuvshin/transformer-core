# TransformerCore



### Описание:

**TransformerCore** — это **плагин для NanoCAD**, который реализует такую деталь, как **сердечник электрического трансформатора** (простыми словами: объёмная буква 'Ш' или 'Е').

Данная работа создана _в рамках учебной практики_ в **ИПМКН НИ ТГУ[^1]** совместно с партнёром **ТомскНИПИнефть[^2]** (руководитель практики от профильной организации/предприятия: _Белошицкий Артём_).

**В проекте реализованы**:
1. Ввод параметров сердечника трансформатора;
2. Валидация введённых параметров (ограничения взяты из здравого смысла + см. комментарий к конструктору _TransformerCoreParameters()_);
3. Построение детали с заданными параметрами в среде NanoCAD.



### Инструменты:

- .NET 6;
- C#;
- WinForms;
- NanoCAD 24 SDK.



### Запуск

1. Соберите проект `TransformerCore.View` (через обычный `dotnet build`, будучи в директории проекта);
2. В консоли NanoCAD введите `NETLOAD`;
3. В окне проводника выберите файл `transformer-core -> TransformerCore.View -> bin -> Debug -> net6.0-windows -> TransformerCore.View.dll`;
4. В консоли NanoCAD введите `CREATETRANSFORMERCORE`;
5. В форме введите запрашиваемые параметры и нажмите кнопку _Готово_;
6. Дождитесь построения чертежа в среде NanoCAD и наслаждайтесь.
